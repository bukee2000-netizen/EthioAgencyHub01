## Employee Task & Activity Tracking - Detailed Fields

### Employee Status Pipeline
- [ ] REGISTERED - Employee created, pending document review
- [ ] DOCUMENT_REVIEW - Documents being reviewed
- [ ] MOLS_PENDING - MOLS sync initiated, awaiting verification
- [ ] INTERVIEW_UPLOADED - Interview video uploaded via Telegram
- [ ] TRAVEL_READY - Travel arranged, tickets booked
- [ ] DEPLOYED - Employee deployed to destination
- [ ] ARCHIVED - Employee archived after deployment cycle

### Document Status Tracking
- [ ] PENDING - Document uploaded, awaiting review
- [ ] VERIFIED - Document approved and verified
- [ ] REJECTED - Document rejected, needs resubmission
- [ ] EXPIRED - Document expired, renewal needed

### Travel Status Tracking
- [ ] SCHEDULED - Travel date set
- [ ] TICKETED - Tickets booked
- [ ] READY - Pre-departure checklist complete
- [ ] DEPARTED - Employee departed
- [ ] ARRIVED - Employee arrived at destination
- [ ] CANCELLED - Travel cancelled

### Visa Application Stages
- [ ] DOCUMENT_COLLECTION - Gathering required documents
- [ ] PORTAL_REGISTRATION - Registered on embassy portal
- [ ] SUBMITTED_TO_EMBASSY - Application submitted
- [ ] VISA_APPROVED - Visa approved
- [ ] REJECTED_CORRECTION - Rejected, corrections needed

### MOLS Stages
- [ ] CONTRACT_LINKED - Employment contract linked
- [ ] MOFA_AUTHENTICATED - Ministry of Foreign Affairs authenticated
- [ ] EMBASSY_LEGALIZATION - Embassy legalization completed
- [ ] MOLS_SUBMITTED - Submitted to MOLS
- [ ] APPROVED - MOLS approval received

### Cross-Match Verification Fields
- [ ] nameMatch - Name matches across documents
- [ ] passportMatch - Passport number matches
- [ ] visaCountryMatch - Visa country matches destination
- [ ] passportExpiryOk - Passport not expired
- [ ] visaExpiryOk - Visa not expired

### Agent Activity Tracking
- [ ] selectedByAgent - Which agent selected the employee
- [ ] selectedAt - Timestamp of selection
- [ ] agentPerformance - Agent metrics and ratings
- [ ] agentCommission - Commission tracking per deployment

### Psychology Assessment Tracking
- [ ] psychologyScore - Assessment score
- [ ] psychologyAnswers - Detailed answers
- [ ] psychologyNotes - Assessor notes

### Employee Reports - Per Employee
- [ ] Registration date and status
- [ ] Document processing timeline
- [ ] MOLS sync history
- [ ] Interview video status
- [ ] Travel arrangements
- [ ] Visa application progress
- [ ] Agent assignments
- [ ] Deployment outcome

### Dashboard KPIs

- [ ] Document pipeline (pending/verified/rejected)
- [ ] Today's departures
- [ ] MOLS pending sync count
